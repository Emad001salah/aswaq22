# قائمة مهام الإطلاق (Deployment Tasks)

- [ ] **1. تجهيز بيئة Git و GitHub**
  - [ ] تهيئة مستودع Git محلياً (إن لم يكن موجوداً).
  - [ ] إنشاء ملف `.gitignore` وتأكيد استبعاد الملفات الحساسة (`.env` وغيرها).
  - [ ] ربط المستودع بحساب GitHub الخاص بك.

- [ ] **2. تجهيز إعدادات النشر السحابي (IaC & Configs)**
  - [ ] إنشاء ملف إعدادات Vercel (`vercel.json`) للواجهة الأمامية.
  - [ ] إنشاء ملف إعدادات Render/Railway (`render.yaml` أو ما شابه) للواجهة الخلفية.

- [ ] **3. إعدادات التكامل المستمر (CI/CD)**
  - [ ] إنشاء مجلد `.github/workflows`.
  - [ ] كتابة ملف `ci.yml` لفحص الكود (Linting, TypeScript Check, Tests).

- [ ] **4. إعداد المراقبة وتتبع الأخطاء (Sentry & Monitoring)**
  - [ ] التثبيت والتحقق من حزم `sentry` للواجهتين الأمامية والخلفية.
  - [ ] تهيئة Sentry في الكود (`main.tsx` و `server.ts`).

- [ ] **5. معالجة الصور والتخزين المنفصل (S3/Cloudflare R2)**
  - [ ] مراجعة دوال رفع الصور في الواجهة الخلفية والتأكد من توافقها مع `aws-sdk/s3`.

- [ ] **6. استكمال المنصات السحابية (بمساعدتك اليدوية)**
  - [ ] إنشاء قاعدة البيانات في Upstash (Redis).
  - [ ] إعداد خادم MeiliSearch.
  - [ ] ربط `aswaq22.com` في Cloudflare.

- [ ] **7. اختبار الأمان والضغط (Security & Load Testing)**
  - [ ] التحقق من إعدادات الـ CORS و Helmet.
  - [ ] تنفيذ اختبارات ضغط أساسية للتأكد من استقرار الخادم.
